# Universal 16S rRNA Primers for Bacterial Identification
# Format: PrimerName ForwardSequence ReverseSequence
#
# These primers target the bacterial 16S ribosomal RNA gene for universal amplification

# 27F/1492R - Nearly full-length 16S rRNA gene (~1,400 bp)
# From https://www.nature.com/articles/s41598-025-10999-8
16S_FullLength AGMGTTYGATYMYGGCTCAG TACGGYTACCTTGTTACGACTT

# V1V2 used by partners
16S_V1V2 AGAGTTTGATCCTGGCTCAG CTGCTGCCTYCCGTA

# 341F/806R - V3-V4 hypervariable regions from Novogene (~465 bp)
# Commonly used in microbiome studies
16S_V3V4_Novogene CCTAYGGGRBGCASCAG GGACTACNNGGGTATCTAAT

# From article which says that 16S V3-V4 primers are bad when human DNA is present
# 16S_V3V4_s41598 TCGTCGGCAGCGTCAGATGTGTATAAGAGACAGCCTACGGGNGGCWGCAG GTCTCGTGGGCTCGGAGATGTGTATAAGAGACAGGACTACHVGGGTATCTAATCC
16S_V3V4_s41598_short CCTACGGGNGGCWGCAG GACTACHVGGGTATCTAATCC

# 515F/806R - V4 hypervariable region (~291 bp)
# Earth Microbiome Project (EMP) standard primers
# Reference: Caporaso et al.
# Note: Also amplifies archaeal 16S (not bacteria-specific)
16S_V4_EMP GTGCCAGCMGCCGCGGTAA GGACTACHVGGGTWTCTAAT
